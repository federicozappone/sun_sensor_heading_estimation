## Quick Start

Install the dependencies:

```
pip install -r requirements.txt
```

Calibrate your camera with any standard procedure and modify the ```sun_sensor.py``` script with the correct values of ```px, py, fx, fy```

Run the script:

```
python sun_sensor.py
```
